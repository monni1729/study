Cell Image Classification
==============================

비교적 초창기에 진행한 연구입니다.
소속 연구실의 특성을 살려 4종류의 암세포 사진 직접을 준비하였습니다.
총 2000장 정도의 이미지를 확보할 수 있었고, 세포의 형태학적 특징으로만 분류하기 위해서 grayscale 이미지로 변환하였습니다.
몇가지 분류 모델을 선정하여 작동 원리와 구현 방법을 공부하였고, 이를 이용하여 모델을 학습하였습니다.
학습 과정에서 이미지의 resolution, augmentation 방법에 따른 accuracy 차이를 관찰하였습니다.
