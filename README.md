# 설명

## cell_image_classification
비교적 과거에 진행한 연구입니다.
소속 연구실의 특성을 살려 4종류의 암세포 사진 직접을 준비하였습니다.
총 2000장 정도의 이미지를 확보할 수 있었고, 세포의 형태학적 특징으로만 분류하기 위해서 grayscale 이미지로 변환하였습니다.
몇가지 분류 모델을 선정하여 작동 원리와 구현 방법을 공부하였고, 이를 이용하여 모델을 학습하였습니다.
학습 과정에서 이미지의 resolution, augmentation 방법에 따른 accuracy 차이를 관찰하였습니다.

## imagenet_classification
회사에서 근무할 때 진행되었던 프로젝트입니다.
ImageNet-21k 데이터셋을 활용한 분류 모델입니다.
비교적 최신 모델이었던 EfficientNet과 ResNeSt 모델을 활용하였고 Pytorch를 사용하여 개발하였습니다.
개발 단계에서는 official code를 가져와 사용하였지만, 신경망 구조를 이해하기 위해 해당 논문을 꼼꼼히 읽어보고
부분적으로 구현해보면서 공부하였습니다.
신경망을 제외한 부분은 모두 본인이 구현 작성하였습니다.

## RL
비교적 과거에 공부했던 내용입니다.
connect4라는 간단한 보드 게임을 환경으로 사용하여 MCTS를 구현하였습니다.
또한, OpenAI의 Gym을 이용하여 Policy_Gradient를 구현하는 연습을 하였습니다.

## face_recognition
회사에서 근무할 때 진행되었던 프로젝트입니다.
얼굴 인식 모델은 기존의 분류 모델과 비교하여 triplet loss, arcface loss와 같은 특별한 loss function을 필요로 합니다.
해당 프로젝트를 진행하면서 loss function에 대해서 조금 더 깊이 생각해볼 기회가 있었습니다.

## coding_test
코딩테스트 문제를 풀었던 코드를 일부 수록하였습니다. 
원하는 알고리즘을 구현할 수 있는 역량을 기르기 위해서 코딩테스트 문제를 풀어보았고, 큰 재미도 느낄 수 있었습니다. 
해당 웹사이트에 수록된 어려운 문제들의 90% 이상을 스스로의 힘으로 풀어내었고 이러한 경험은 이후에 코드를 이해하고 작성하는 과정에 큰 도움을 주었습니다.
