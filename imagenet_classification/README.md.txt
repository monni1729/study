ImageNet-21k Classification
==============================

회사에서 근무할 때 진행되었던 프로젝트입니다.
ImageNet-21k 데이터셋을 활용한 분류 모델입니다.
비교적 최신 모델이었던 EfficientNet과 ResNeSt 모델을 활용하였고 Pytorch를 사용하여 개발하였습니다.
개발 단계에서는 official code를 가져와 사용하였지만, 신경망 구조를 이해하기 위해 해당 논문을 꼼꼼히 읽어보고
부분적으로 구현해보면서 공부하였습니다.
학습 데이터의 용량이 1TB 이상이며 학습 시간을 단축하기 위해 albumentation 라이브러리를 사용하였습니다.
